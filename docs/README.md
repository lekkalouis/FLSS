# FLSS Documentation Index

Documentation is organized by audience:

- `operator-docs.md` – dispatch and warehouse runbooks.
- `admin-docs.md` – governance and platform administration.
- `dev-docs.md` – engineering architecture and workflows.
- `database-and-remote-access.md` – PostgreSQL schema and public tunnel setup runbook.
- `rpi4-full-system-runbook.md` – full FLSS deployment on a Raspberry Pi 4 (backend + frontend + service setup).
- `raspberry-pi-physical-station.md` – GPIO/camera-based physical station implementations.

Supporting deep-dive notes remain in this folder for specific modules and initiatives.
- `feature-module-upgrade-map.md` – inventory of core/add-on modules and upgrade opportunities.
